
## Special Clauses

| 2                                                                                                       | Addendum Copy      |                                                                                                           |
| The Grantee must have a copy of this Addendum onsite of construction at all times during their          |                    |                                                                                                           |

| 3                                                                                                       | Additional Charges |                                                                                                           |
| Additional charges will apply if Grantor's annual surface lease rentals are increased as a result of    |                    |                                                                                                           |
| Grantee's activity under this Agreement.                                                                |                    |                                                                                                           |
| 4                                                                                                       | Affiliates         |                                                                                                           |
| The Master Road Use Agreement shall be binding upon the Grantor and any of its affiliates or            |                    |                                                                                                           |
| subsidiaries and their respective successors (by reason of amalgamation or name change) and the         |                    |                                                                                                           |
| Grantee. The Grantor agrees to provide notice to the Grantee within a reasonable time of any            |                    |                                                                                                           |
| amalgamation or name change affecting the Grantor.                                                      |                    |                                                                                                           |
| 5                                                                                                       | Expiry             |                                                                                                           |
| In addition to clause 20.1 on CAPLA's Version III Master Road Use Agreement, the expiry date of this    |                    |                                                                                                           |

| operations/construction has not commenced.                                                              |                    |                                                                                                           |
| 6                                                                                                       | Gates              |                                                                                                           |

| contact Grantor's field representative to discuss these operations so there is no confusion as to what  |                    |                                                                                                           |
| the expectations are. If the Grantee wants to amend these operations with the AER then the Grantee      |                    |                                                                                                           |
| must discuss their proposal with Grantor's field representative and it must be approved by all parties  |                    |                                                                                                           |
| involved.                                                                                               |                    |                                                                                                           |
| 7                                                                                                       | Grantee            |                                                                                                           |
| If the Grantee is going to have excessive traffic then they may be requested to provide some kind of    |                    |                                                                                                           |

| date of the addendum and the grantee will be invoiced monthly thereafter. For all Wellsite road use     |                    |                                                                                                           |
| addendums, maintenance fees charged as per the road use fee schedule shall begin to accrue as per       |                    |                                                                                                           |
| the spud date of each well and the Grantee will be invoiced monthly thereafter.                         |                    |                                                                                                           |

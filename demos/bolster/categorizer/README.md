## Getting Started

First, run the LLM server:

./Meta-Llama

Then, run the api.py in the bolster directory:

cd ?/demos/bolster
python api.py

Finally run the frontend:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or, if ur really the goat
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
